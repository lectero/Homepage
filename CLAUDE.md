# Homepage — Lectero Official Website

## Stack
- **Framework:** Astro 6
- **Styling:** Tailwind CSS 4
- **Interaktivitet:** React (via Astro islands)
- **Hosting:** Cloudflare Pages (planerad)

## Kommandon
```bash
npm run dev      # Starta dev-server
npm run build    # Bygg för produktion
npm run preview  # Förhandsgranska bygget
```

## Konventioner
- Svenska UI-strängar, engelsk kod
- Astro-komponenter (.astro) för statiskt innehåll
- React-komponenter (.tsx) bara där interaktivitet behövs (formulär, animationer)
- Tailwind utility classes — inga separata CSS-filer utöver global.css
