# Feltaxonomi (K och S)

## Kodformat

`<DOMÄN>.<GRUPP>.<KOD>`

Exempel:
- `K.CONFIG.MISSING`
- `K.CONNECTIVITY.SERVER_UNREACHABLE`
- `S.AUTH.DENIED`
- `S.EXTERNAL.LLM_TIMEOUT`
- `S.LOGIC.STATE_INCONSISTENT`
