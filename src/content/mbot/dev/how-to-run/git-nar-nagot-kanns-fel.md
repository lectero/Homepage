# Git – när något känns fel

Om något känns osäkert, stanna. Push inte. Gissa inte.

## Vanliga situationer

### Jag ser inte mina ändringar
Titta i GitHub Desktop under Changes. Har du committat?

### Andra ser inte mina ändringar
Har du pushat?

### GitHub är tomt
Filerna är inte committade i rätt repo, eller så pushar du fel remote.

### Jag får ett konstigt fel
Läs exakt vad som står. Ta en skärmdump. Fråga.

## Säkra åtgärder
- titta i GitHub Desktop
- avbryt operationer
- stäng Desktop och öppna igen
- avstå från push

## Undvik när du är osäker
- force push
- radera .git
- “snabbfixar” utan förståelse

## Princip
Inget i Git är akut. Det värsta som händer om du väntar är ofta ingenting.
