# Kort intro till Git

Git används för att versionssätta dokumentation och webbsidor i HMB Mailbot.

## Grundidé
Du arbetar alltid lokalt och delar ändringar först när du själv väljer det.

## Grundflöde
1. Pull när du börjar
2. Arbeta lokalt
3. Commit när något är värt att spara
4. Push för att dela

Git ersätter traditionell fil-låsning.
