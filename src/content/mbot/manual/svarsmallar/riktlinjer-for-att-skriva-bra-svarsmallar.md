# Riktlinjer för att skriva bra svarsmallar

Detta kapitel beskriver hur svarsmallar i Mailbot ska utformas för att fungera som tillförlitliga, kontrollerade och återanvändbara byggstenar i ett AI-stött e-postflöde. Texten är avsedd både som praktiskt stöd vid mallskrivning och som kvalitetsreferens för att förstå varför vissa mallar fungerar bättre än andra i ett system där mänskligt ansvar alltid är överordnat automatisering.
