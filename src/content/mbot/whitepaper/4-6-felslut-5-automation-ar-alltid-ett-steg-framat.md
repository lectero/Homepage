# 4.6 Felslut 5: ”Automation är alltid ett steg framåt”

Ett mer subtilt felslut är den normativa föreställningen att automation per definition representerar framsteg. I denna logik blir varje steg bort från mänsklig inblandning ett tecken på teknisk mognad.

Historiskt sett är detta påstående svagt underbyggt. Studier av automatisering i flyg, sjukvård och industri visar att överautomation ofta leder till:

- minskad situationsmedvetenhet

- sämre förmåga att hantera avvikelser

- ökad sårbarhet vid systemfel

I servicekommunikation innebär detta att ett system som ”klarar sig självt” i normallägen kan vara sämre rustat för de situationer där organisationens omdöme faktiskt prövas.

**”Mognad är att veta när man inte ska automatisera**

Ett moget system är inte det som automatiserar mest,  
utan det som automatiserar rätt saker.”
