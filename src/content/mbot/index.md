# HMB Mailbot

![](/mbot/assets/images/icon_256x256.png)

<div class="hero">

HMB Mailbot är ett system för kontrollerad, AI-stödd e-posthantering, byggt för verksamheter där korrekthet, spårbarhet och ansvar är viktigare än maximal automation.

<p class="lead">
Mailboten läser inkommande e-post i Microsoft 365, klassificerar ärenden, föreslår svar som drafts, och hjälper kansliet att arbeta konsekvent utan att tappa mänsklig kontroll.
</p>
