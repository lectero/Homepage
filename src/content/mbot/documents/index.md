# Dokument

Här kan du lägga material som whitepaper, manual, och annan referens. Du kan välja mellan att länka till PDF eller att konvertera till Markdown.

## Rekommenderad struktur

- `docs/documents/` för indexsidor och korta introduktioner
- `docs/assets/pdfs/` för PDF som ska länkas
- `docs/assets/images/` för bilder till sidorna

## Varför Markdown för manual

Manual i Markdown ger:

- sökbarhet i hela innehållet
- interna länkar mellan avsnitt
- enklare diff och versionshantering

PDF är bra som distribution, men svag som arbetsformat.

## Whitepaper

Lägg PDF här:

- `docs/assets/pdfs/whitepaper.pdf`

och länka från sidan nedan.

## Manual

Två alternativ:

1. Behåll PDF, lägg den i `docs/assets/pdfs/manual.pdf`
2. Gör om till Markdown, och låt PDF vara en export

Sidan nedan är förberedd för båda alternativen.
