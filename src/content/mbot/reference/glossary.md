# Ordförklaringar

**Draft**  
Ett förslag till svar som kan granskas innan det skickas.

**Taxonomi**  
En uppsättning kategorier och underkategorier som beskriver vilka typer av ärenden som kommer in.

**Policy**  
Regler och trösklar som avgör hur ett ärende får hanteras.

**Idempotens**  
Att samma input kan behandlas flera gånger utan att resultatet blir dubbelt, till exempel inga dubbla drafts.

**Triangulering**  
Att ta flera bedömningar för att minska osäkerhet, på bekostnad av mer kostnad.
